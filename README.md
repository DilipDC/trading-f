## ✨ Features

| Category | Features |
|----------|----------|
| 📊 **Stock Dashboard** | Live prices, % change, auto-refresh (polling/WebSocket) |
| 📈 **Chart System** | Candlestick / Line toggle, timeframes (1m,5m,1D), smooth animations |
| 🕒 **Market Status** | Indian market hours (9:15–15:30), countdown timer |
| ⭐ **Watchlist** | Add/remove stocks, localStorage persistence |
| 💰 **Deposit System** | Min ₹100, QR code modal, status tracking |
| 💸 **Withdraw System** | UPI ID, name, amount validation, API calls |
| 🛒 **Order Panel** | Buy/Sell toggle, Market/Limit orders, estimated cost |
| 📁 **Portfolio Panel** | Balance, P&L, holdings list |
| 🎨 **UI/UX** | Dark theme (Zerodha/TradingView style), fully responsive |
| 🔊 **Sound System** | Click, success, error sounds |
| 🔔 **Notifications** | Toast messages for success/error/info |
| ⚡ **Performance** | Optimized re-renders, lazy loading, debouncing |

---

## 🖥️ Tech Stack

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: CSS3 (custom dark theme, animations)
- **Charts**: Chart.js / TradingView lightweight
- **State Management**: React hooks + localStorage
- **API Layer**: Fetch + WebSocket ready
- **Icons**: Font Awesome 6
- **Sounds**: HTML5 Audio
